Fama-French 風の日本 5 ファクター(Mkt-RF / SMB / HML / RMW / CMA)を題材にしたファクター戦略シミュレーター。`Japan_5_Factors.csv` をロードし、`quant_engine.py` で戦略構築、Vercel 経由でも軽量 UI を配信。

主要機能

  • `Japan_5_Factors.csv` 同梱で即座にローカル動作
  • `quant_engine.py` でファクター戦略の組み立て
  • `universe_manager.py` で銘柄ユニバース管理
  • `data_provider.py` でデータソース統一
  • Vercel `api/index.py` 経由で軽量 web 配信
